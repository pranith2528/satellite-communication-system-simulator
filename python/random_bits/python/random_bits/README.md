# Random Binary Generator

## Objective

Generate random binary bits for digital communication system simulations.

## Theory

In digital communication systems, information is represented as binary bits (0s and 1s). Since real communication data is not always available during simulations, pseudo-random binary sequences are generated.

## Inputs

- Number of bits (default: 1000)

## Outputs

- Random bit sequence
- Number of zeros
- Number of ones
- Percentage of zeros
- Percentage of ones
- Distribution plot
